# Food Tier List

Hello there!

This is my first bigger front-end project: a **Food Tier List** built **only with HTML and CSS**.

I made it to practice what I'm learning and because arguing about which food belongs in S-tier is always funny

---

## About the Project

The goal was to create something visually nice and interactive without using JavaScript. 

It features:
- A clean dark-themed design
- A classic tier list with S, A, B, C, D, and F tiers
- Food cards with emojis
- Smooth hover effects
- A fun "Generate craving" button

Everything was built from scratch using only HTML and CSS so I could really understand how things work.

## What’s Inside

- Modern dark layout
- Vertical tier list (S to F)
- Responsive food cards with hover animations

Even though it’s a simple project, I’m really proud of how it turned out visually.

## Technologies Used

- HTML5
- Pure CSS
  - Flexbox
  - CSS Grid
  - Transitions & hover effects

## How to Run

It’s super easy:

1. Download or clone this repository
2. Open the `index.html` file in your browser
3. Enjoy the tier list

No installation needed.
